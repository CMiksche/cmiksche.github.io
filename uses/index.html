Uses

Software⌗

I have iSH on my iPhone which makes it possible to do some small changes on the go. Aside from that, my MacBook has macOS and I mostly code in VS Code but also in some JetBrains IDEs like PyCharm.

My most used computer languages and frameworks include: Angular, React, Vue.js, TypeScript, HTML, CSS, GoLang, Python, Kotlin, Java and Rust but this should not be an CV. (I also have experience with a lot more technologies and systems)

My PC runs on Win 11 and there I also use the same IDEs. I have various single-board computers and older devices, which all run on either some form of Linux or BSD. In the past I dual-booted my Desktop PCs and Laptops with Linux or FreeBSD, but now my main usage of *nix is mostly limited to WSL on Windows, the macOS command line and multiple VPS.

Hardware⌗

Mobile⌗

I have a collection of various Laptops and Mobile Phones, but currently I use:

  • MacBook Air M1 2020 with 16 GB Memory and 500 GB of disk space
  • iPhone 12 mini

My older devices are being used to test my software on other operating systems and architectures. Currently e.g. Android 10 on a Mi A2 Lite and Arch Linux on an Acer Laptop from 2013.

Cameras⌗

I am the proud owner of a Canon EOS 1200D which is my main DSLR, but most of the time I just use my iPhone.
